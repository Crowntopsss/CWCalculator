# Installation
Clone the repository
```sh
$ git clone https://github.com/Crowntopsss/CWCalculator.git
$ cd CWCalculator
$ pod install
```


Project was done in:
* Xcode Version Version 13
* Swift 4
* iOS Deployment target 14
